> Install dependencies using [pnpm](https://pnpm.io)

## `backend`
<details>
<summary>
How to start
</summary>

Unzip packages.zip to ./packages
```sh
cd ./backend
tar -xf packages.zip
```

pnpm install
```sh
pnpm i
```

copy env file and set correct values on .env
```sh
copy .env.example .env
```

start the backend server
```sh
nodemon
```

</details>

## `frontend`
<details>
<summary>
How to start
</summary>

npm install
```sh
cd ./client
npm i
```

copy env file
```sh
copy .env.example .env
```

```sh
npm start
```
